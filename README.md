# KennyLoggingsBot

Kenny Loggings fetches the price of your favorite cryptocurrencies from CoinGecko.

The bot is available on https://t.me/KennyLogginsBot.

## Commands

- `/bnb`
- `/btc`
- `/dnd`
- `/donate`
- `/eth`
- `/steel`
- `/titan`
